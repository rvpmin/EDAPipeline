# Pipeline EDA - Registro Nacional de Cáncer

## Descripción
Este proyecto implementa un pipeline automatizado de análisis exploratorio de datos (EDA) sobre una base de datos del Registro Nacional de Cáncer.

## Estructura
- data/: datos (no incluidos)
- scripts/: código fuente
- out/: resultados
- main.sh: ejecuta el pipeline
